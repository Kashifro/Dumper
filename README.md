# Dumper5000

RTTI based vtable extraction plugin

## What it does

* virtual methods
* approx structs
* pseudocodes
* multiple inheritance vtables
* compare two vtables

### IDA Side

* creates vtable structs
* creates class structs
* assigns function pointer types
* updates local type database

### Required

* RTTI present in target binary

## Usage

Launch plugin:

```text
Ctrl + Shift + V
```
